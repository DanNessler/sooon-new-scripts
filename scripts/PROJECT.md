# sooon Project Context

## What this is
Mobile-first concert discovery platform. Webflow CMS, full-screen snap-scroll feed, audio previews.

## Key constraints
- Webflow only (no frameworks)
- iOS Safari is primary target
- Finsweet Attributes v2 in use
- Client-First class structure

## Current custom scripts
- sooon-head.js: Asset loader, eager card limiting
- sooon-footer.js: Audio control, modal scroll lock, filter navigation
